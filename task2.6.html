Tag Description
<html>...</html> Declare the webpage to be written in HTML.
<head>...</head> Delimits the page's head.
<title>...</title> Defines the titles(not displayed on the page).
<body>...</body> Delimits the page's body.
<b>...</b> Set in boldface.
<i>...</i> Set in italic.
<center>...</center> Center the text/images or anything on the page horizontally.
<ul>...</ul> Brackets an unordered(bulleted) list.
<ol>...</ol> Brackets a numbered list.
<li>...</li> Brackets an item in an ordered or unordered list.
